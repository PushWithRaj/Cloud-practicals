# Cloud Computing Lab - 10 Practicals

This repository contains a comprehensive set of 10 practical exercises covering public cloud administration (AWS) and private cloud management (OpenStack). 

## 🚀 Overview
The lab covers essential cloud concepts including Infrastructure as a Service (IaaS), networking, auto-scaling, monitoring, and security.

## 📁 Practical Index

| No. | Practical Title | Key Services Used |
|:---|:---|:---|
| 01 | Introduction to Cloud Platforms | AWS, Azure, GCP Consoles |
| 02 | Launching Amazon EC2 Instance | EC2, Key Pairs, Security Groups |
| 03 | Setting Up a Custom VPC | VPC, Subnets, IGW, NAT Gateway |
| 04 | Auto Scaling & Load Balancing | ALB, ASG, Scaling Policies |
| 05 | Static Website Hosting | Amazon S3, Bucket Policies |
| 06 | Cloud Monitoring & Alerts | CloudWatch, SNS, Alarms |
| 07 | OpenStack Installation | DevStack, Linux, IaaS |
| 08 | OpenStack Instance Lifecycle | Glance, Nova, Keystone |
| 09 | OpenStack Networking | Neutron, Routers, Floating IPs |
| 10 | Cloud Security & IAM | IAM Roles, SSE-S3, Firewalls |

## 🛠️ Requirements
* Active AWS Free Tier Account
* Ubuntu 22.04 LTS (for OpenStack labs)
* SSH Client (PuTTY or Terminal)
* Basic Linux CLI knowledge

## 📖 Structure
Each directory contains:
* **Theory:** Background of the cloud service.
* **Procedure:** Step-by-step execution guide.
* **Diagrams:** Flowcharts and architectural logic.
* **Scripts:** Any code or stress-test scripts used.
